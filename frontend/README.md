# Ride Share App Frontend

A Vue.js frontend for a ride-sharing application.

## Technologies Used

*   [Vue.js](https://vuejs.org/)
*   [Vue Router](https://router.vuejs.org/) for routing
*   [Pinia](https://pinia.vuejs.org/) for state management
*   [Tailwind CSS](https://tailwindcss.com/) for styling
*   [Leaflet](https://leafletjs.com/) for maps and routing
*   [Vite](https://vitejs.dev/) as a build tool

## Features

*   User authentication
*   Requesting and managing trips
*   Driver and passenger views

## Getting Started

### Prerequisites

*   Node.js (version ^20.19.0 || >=22.12.0)
*   A running instance of the backend server.

### Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the application

```bash
npm run dev
```
